# 快速开始指南

## 🚀 一分钟上手

### 1. 启动应用

应用已经在运行！访问：http://localhost:3000

### 2. 测试功能

尝试使用这个有字幕的B站视频测试：

```
https://www.bilibili.com/video/BV1xx411c7mD
```

或任何其他有CC字幕的B站视频链接。

### 3. 配置AI功能（可选）

如果您想使用AI生成更智能的笔记：

1. 获取阿里云通义千问API密钥：https://dashscope.console.aliyun.com/apiKey
2. 编辑 `.env.local` 文件
3. 添加：`QWEN_API_KEY=你的API密钥`
4. 重启开发服务器

## 📝 使用流程

1. 打开应用 → http://localhost:3000
2. 粘贴B站视频链接
3. 点击"开始提取"
4. 等待处理完成（15-30秒）
5. 查看生成的笔记
6. 复制或下载笔记

## ⚠️ 注意事项

- ✅ 只支持有CC字幕的视频
- ✅ 没配置API密钥会使用简单格式（依然可用）
- ✅ 支持深色模式
- ✅ 移动端友好

## 🔧 常见问题

**Q: 提示"该视频暂无字幕"怎么办？**  
A: 换一个有CC字幕的视频试试。未来版本会支持语音转文字。

**Q: AI生成失败怎么办？**  
A: 系统会自动使用简单格式，内容依然可用。

**Q: 如何部署到线上？**  
A: 参考 DEPLOYMENT.md 文档，推荐使用Vercel一键部署。
